## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Oxunjon0420/Portfolio-Oxunjon.git
cd Portfolio-Oxunjon
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
